<li>Create and return a Tuple -

<li>Access Elements - 
  Given the tuple: your_tuple = ("apple", "banana", "cherry", "date")<br>

</li>

<li>Check for Membership - 
  Write a Python program that checks if the number 25 is present in a tuple.<br>
